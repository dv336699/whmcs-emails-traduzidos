This is a reserved area for the automatic e-mails sending via WHMCS Admin Panel.
To install and customize your WHMCS Email, go to Setup > General Settings > Mail tab.

Now, go to the Antler downloading folder and find the header.html, footer.html and global.css files inside the email folder. After, replace the HTML and CSS code on the following fields:

• Global Email CSS Styling;
• Global Email Header Content;
• Global Email Footer Content.

Please Note - The email templates allow you to customise the messages that go out to your customers when actions occur inside WHMCS. To customise an email template, go to Setup > Email Templates and click the edit icon next to the template you want to change.

To learn more about email templates check official page or go to Documentation folder.
